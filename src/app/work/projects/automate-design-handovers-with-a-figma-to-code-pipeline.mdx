## Overview

Landing page para el negocio de fisioterapia Heals, desarrollada con Next.js. Incluye consultas personalizadas y optimización SEO para mejorar la visibilidad y captar clientes.

## Características principales

- **Desarrollo con Next.js**: Aplicación web rápida y optimizada para SEO.
- **Consultas personalizadas**: Formulario de contacto y consultas para clientes.
- **Optimización SEO**: Estrategias para mejorar el posicionamiento en buscadores y aumentar la captación de clientes.
- **Diseño moderno y responsivo**: Interfaz atractiva y adaptable a cualquier dispositivo.

## Tecnologías utilizadas

- **Next.js**: Framework principal para el desarrollo web.
- **Tailwind CSS**: Para estilos rápidos y personalizables.
- **Vercel**: Para despliegue y hosting.
- **GitHub**: [https://github.com/Nickr010119](https://github.com/Nickr010119) Control de versiones y colaboración.

## Resultado

La landing page facilita la captación de clientes y la gestión de consultas para el negocio de fisioterapia, mejorando la presencia online y el rendimiento en buscadores.
