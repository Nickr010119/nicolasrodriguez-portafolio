## Overview

Marketplace para el sector de construcción desarrollado con Next.js. Permite a empresas y profesionales publicar, buscar y gestionar proyectos de construcción de manera eficiente.

## Características principales

- **Marketplace especializado**: Publicación y gestión de proyectos, productos y servicios para el sector de construcción.
- **Desarrollado con Next.js**: Aplicación web rápida, escalable y optimizada para SEO.
- **Panel de administración**: Gestión de usuarios, proyectos y transacciones (Futuro).
- **Integración con Vercel**: Despliegue y previsualización continua.
- **Diseño moderno y responsivo**: Interfaz intuitiva y adaptable a cualquier dispositivo.

## Tecnologías utilizadas

- **Next.js**: Framework principal para el desarrollo web.
- **Tailwind CSS**: Para estilos rápidos y personalizables.
- **Vercel**: Para despliegue y hosting.
- **GitHub**: [https://github.com/Nickr010119](https://github.com/Nickr010119) Control de versiones y colaboración.

## Resultado

El marketplace facilita la conexión entre empresas y profesionales del sector de construcción, optimizando procesos y mejorando la visibilidad de los proyectos. El uso de Next.js y Vercel permite un desarrollo ágil y un rendimiento óptimo.
